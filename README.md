# Project Information Integration WS25/26

This project is part of the course "Information and Data Integration" at the FSU Jena in the winter semester 2025/2026. The project focuses on integrating various data sources to create a unified dataset.

To run the project, follow these steps:

1. Clone the repository to your local machine.
2. Install Python 3.14 and the required dependencies listed in the `requirements.txt` file.
3. Run the main script to start the data integration process: `python3 -m src.main`.

The project is structured as follows:
- `src/`
    - `integration_scripts/`: Contains the scripts responsible for integrating different data sources. Check the `README.md` files for more details.
    - `utils/`: Contains utility functions and helper scripts used across the project. These are used for data normalization, schema mapping, and other common tasks.
    - `main.py`: The main entry point for the project, which orchestrates the data integration process.
- `data/`:
    - `raw/`: Contains the raw data files that are used as input for the integration process.
    - `processed/`: Contains the processed and integrated datasets that are generated by the project.
